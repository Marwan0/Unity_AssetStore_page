# SolidPixel Studio Landing Page

A modern, professional landing page for SolidPixel Studio - Production-grade Unity assets.

## Features

- Clean, production-ready design
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation
- Fade-in animations on scroll
- Professional color scheme
- Fast loading and optimized

## Getting Started

Simply open `index.html` in your web browser or serve it through a web server.

### Local Development

You can use any local server:

```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server)
npx http-server

# PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## File Structure

```
.
├── index.html      # Main HTML structure
├── styles.css      # All styling
├── script.js       # Interactive features
└── README.md       # This file
```

## Customization

- **Colors**: Edit CSS variables in `:root` section of `styles.css`
- **Content**: Modify the HTML in `index.html`
- **Animations**: Adjust JavaScript in `script.js`

## Browser Support

Modern browsers (Chrome, Firefox, Safari, Edge) with ES6+ support.
